# Task 1: simple-calculator (Tallinn Polytechnic School)

## Task description
Build a simple calculator (Windows Form App, class calculation)

### Data
- Two numbers
- Arithmetic operator - addition, subtraction, multiplication, division

### Result
- Answer - an arithmetic expression

### Error message
Give an error message if:
- both data fields are blank
- numbers are separated by periods, not commas
- operation is not selected (RadioButton)
- there are entered letters, not numbers
- zero is entered in at least one field (dividing by zero is prohibited)

## Ülesande kirjeldus
Koostage lihtne kalkulaator (Windows Form App, klass arvutamine)

### Andmed
- kaks numbrit
- aritmeetilise operaator – liitmine, lahutamine, korrutamine, jagamine

### Tulemus 
- Vastus – aritmeetiline avaldis

### Veateade 
Väljasta veateade kui:
- mõlemad andmeväljad on tühjad 
-	numbrid on eraldatud punktiga, mitte komaga 
-	tehet ei ole valitud (RadioButton) 
-	sisestatud on tähed, mitte numbrid 
-	vähemalt ühele väljale on sisestatud null (nulliga jagamine on keelatud)
